# Rad-SAC Agent v0.1.0

First public research release of  Beam Weaver.

## Included in this release

- custom hybrid Soft Actor–Critic implementation
- water-photon transport environment
- Monte Carlo sampling utilities for photon interactions
- physics-head pretraining workflow
- phased curriculum training logic
- evaluation utilities for track and PDD comparison
- repository metadata for citation and Zenodo archiving

## Scope

This release is a proof of concept intended for research and reproducibility work in radiation transport and reinforcement learning.
